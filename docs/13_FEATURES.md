# 13. 기능 카탈로그 (현행)

## 인증
- 이메일/비밀번호 회원가입(닉네임 + 선택 프로필 사진) + **이메일 인증**(인증 전 로그인 차단).
- **소셜 로그인** Google/Kakao/Naver — 자체 OAuth(인가코드 흐름). 소셜 가입자는 이메일 인증 생략.
- 세션: `jose` JWT(HS256) → httpOnly 쿠키 `pp_session`(7일). `src/proxy.ts`가 `/dashboard`·`/works`·`/profile` 보호.
- 인증 메일은 SMTP(Gmail) 또는 Resend로 발송(설정 분기, `14_DEPLOYMENT` 참조).

## 프로필 / 작가 홈
- `/profile`: 닉네임, **프로필 사진**, **배경 사진**, **소개(bio)** 수정. 저장 시 세션 재발급(헤더 즉시 반영).
- 공개 **작가 홈** `/author/[id]`: 배경·아바타·닉네임·소개 + 공개 작품 목록(커버 카드).
- 헤더 아바타 → 프로필로 이동.

## 작품(프로젝트) / 회차(Papers)
- **작품 생성과 글쓰기 분리**: `/works/new`에서 작품 메타(제목·소개·장르·태그·**번역 언어 다중 선택**·메인 이미지)만 만들고,
  회차는 작품 안에서 추가. (`/works/new`는 "기존 작품에 글쓰기 / 새 작품 만들기" 선택형)
- 회차: **자유 제목** + **종류(카테고리)** 칩(본편/외전/짧은 컷/낙서/작업 비하인드) + 메인 이미지 + 공개 토글.
- 작품 상세에서 회차 목록(번역 진행 N/M, 공개 상태), 회차 보기(원문/언어별 번역 비교), 수정/삭제.
- 작품/회차 삭제(소유자, cascade).

## 리치 텍스트 에디터 (Tiptap)
- 볼드/기울임/밑줄/취소선, 제목 H1~H3, 목록, 인용구, **이미지 삽입**.
- **이미지 크기 조절**(25/50/75/꽉 채우기/원본), 가운데 정렬.
- 저장 시 서버에서 **sanitize**(허용 태그/속성만; `style`은 width/height만). 사용자 입력·LLM 출력 모두.

## 이미지 / 웹툰 최적화
- 작품/회차 **메인 이미지**(없으면 기본 이미지: 브랜드 그라데이션 ✈).
- 본문 이미지는 반응형 전체 너비(세로로 긴 웹툰도 비율 유지), **lazy 로딩** 자동 주입.
- 저장소: Vercel **Blob**(배포) / `public/uploads`(로컬). `BLOB_READ_WRITE_TOKEN` 유무로 분기.
  Blob 스토어는 **Public** 이어야 함(공개 URL 필요).

## 번역 (보조 기능)
- 작품마다 **여러 대상 언어** 선택. 회차 저장/수정·언어 변경 시 **모든 대상 언어로 번역**.
- **비동기**: 저장 즉시 pending으로 응답, 실제 LLM 번역은 응답 후 백그라운드(`next/server` `after()`).
  진행 중이면 화면이 자동 새로고침(`TranslationPoller`).
- 독자 뷰어는 **원문이 기본**, 번역이 있으면 **언어 버튼**으로 전환(`원문 · 한국어 / English / 日本語…`).
- 언어 추가 시 기존 회차 백필, 제거 시 해당 번역 정리.
- 제공자: Upstage Solar(OpenAI 호환). `.env`의 `LLM_BASE_URL/MODEL/API_KEY`로 교체(Groq/Gemini/Ollama 가능).

## 독자 뷰어
- `/read/[slug]`: 작품 헤더(커버) + **회차 목록**(카테고리 칩·커버 썸네일·댓글 수) + 회차 단위 공유.
- `/read/[slug]/[order]`: 회차 본문(원문/번역 토글) + 작가 배지(작가 홈 링크) + **공유 버튼** + 이전/다음 회차 + 댓글.
- 조회수/유입(referrer·UTM) 기록(IP 해시).

## 공유 / OG
- **작품 단위 공유**(공개 링크 복사) + **회차 단위 공유**(회차 보기·목록·독자 페이지의 공유 버튼).
- **Open Graph / Twitter Card**: 루트 기본값 + 작품/회차/작가 홈마다 동적 제목·설명·이미지.
  - og:image는 **회차/작품 커버 → 본문 첫 이미지 → 작가 배경/아바타 → 브랜드 기본 이미지** 순.
  - 기본 OG 이미지는 `next/og`로 생성(`/opengraph-image`).
- 파비콘 `src/app/icon.svg`(브랜드 종이비행기).

## 댓글
- **회차 단위** 댓글 + **답글(1단계)**. **작성은 로그인 필수**(계정 연동, 세션 닉네임/아바타 표시).
- 삭제: 창작자는 자기 작품 모든 댓글, **작성자 본인**은 자기 댓글. (레거시 익명 댓글은 읽기전용으로 유지.)
- HTML 제거 sanitize. 공개된 회차에만 작성.

## 독자(팬) 기능
- **독자 로그인**: 창작자 로그인/회원가입과 동일(계정 유형 구분 없음). 모든 유저가 읽고·쓰고·작가가 될 수 있음.
- **작가 구독**: 작가 홈에서 구독/해제. 구독자 수 표시. 피드에 반영.
- **하트**: 작가 홈(작가 하트)·작품(작품 하트)에 좋아요. 구독과 별개. 낙관적 토글.
- **독자 피드** `/feed`(로그인 필수): 구독 작가 최근 공개 작품 우선 + 트렌딩(조회·하트) 보강.
- **작가 SNS 링크**: 프로필에서 등록(최대 8). 작가 홈·회차 끝에 버튼 노출.
  클릭은 `/api/links/[id]/go`로 추적(`LinkClick`) 후 리다이렉트 — 서비스 어필용 클릭률 지표.

## 대시보드
- 총 조회/오늘 조회/총 댓글/공개 작품 수, 작품별 조회·댓글, 주요 유입 경로, 최근 댓글, **내 SNS 클릭 요약**.

## 개발자(관리자) 대시보드
- `/admin` — `ADMIN_EMAILS` 환경변수 허용목록만 접근(미포함 시 404).
- 전역 지표(사용자/작품/조회/구독/하트/**SNS 클릭수·클릭률**), 상위 SNS 링크, 플랫폼별 클릭.
